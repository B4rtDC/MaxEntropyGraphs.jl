# Paper figures

Figures embedded by `paper.md` (JOSS submission).

## `schematic.pdf` — package overview (`fig:schematic`)

Source: [`schematic.mmd`](schematic.mmd) (Mermaid flowchart) + [`mermaid-config.json`](mermaid-config.json).
Convention: **black** = core functionality of the package, **blue** = interaction with external packages.

Rebuild (uses the official [mermaid-cli](https://github.com/mermaid-js/mermaid-cli) docker image; no local
node.js required — the config file sets the security level for the HTML table label and the label/arrowhead CSS):

```bash
cd figures
docker run --rm -u "$(id -u):$(id -g)" -v "$PWD:/data" minlag/mermaid-cli \
    -c mermaid-config.json -i schematic.mmd -o schematic.pdf --pdfFit -b white
```

For a quick raster preview, replace the output with `-o schematic.png -s 2`.
